Tag #1 - Firmware Version
Type: undefined[4]
Example: 0200
Description: The firmware version of the camera.
Tag #2 - ISO
Type: short[2]
Example: 0 400
Description: The ISO settings used to take the photo. The first number can be 0 or 1 and could be related to the Auto ISO setting.
Tag #3 - Color Mode (?)
Type: ascii
Example: COLOR
Description: The value is always COLOR. To check on D200, which has a b/w mode and probably uses a new value. Allowed values are:
  • COLOR: The photo has been taken in color mode.
Tag #4 - Quality
Type: ascii
Example: RAW
Description: The quality setting used to take the photo. Allowed values are:
  • RAW: The quality setting was RAW
  • RAW2.7M: The quality setting was RAW (only on model XXX)

  • FLASH: The white balance settings on the camera was "Flash"
  • INCANDESCENT: The white balance settings on the camera was "Incandescent"
  • PRESET:
  • PRESET1:
  • SUNNY: The white balance settings on the camera was "Sunny"
  • WHITE PRESET:
Tag #6 - Sharpening

  • LOW: The photo was taken with a "Low" sharpening.
  • MED.H: The photo was taken with a "Medium Low" sharpening.
  • MED.L: The photo was taken with a "Medium High" sharpening.

Tag #8 - Flash Setting

Tag #171 - Vari Program
Type: ascii
Example: AUTO
Description: For camera models that support it, this tag contains information about the automatic program that was used. Allowed values are:
  • AUTO: The "P" mode was used.
  • CLOSE UP: The "Close up" program was used.
  • LAND SCAPE: The "Landscape" program was used.
  • NIGHT PORTRAIT: The "Night portrait" program was used.
  • NIGHT SCENE: The "Night scene" program was used.
  • PORTRAIT: The "Portrait" program was used.
  • SPORT: The "Sport" program was used.
  • empty: No automatic program was used.
