Change Log

v1.0.RC5 - 18/Feb/2006

  • Fixed JRW-105 Resources are not reset in RAWImageReaderSupport#setInput
  • Fixed JRW-106 getWidth(), getHeight(), getThumbnailWidth(), getThumbnailHeight(), readThumbnail() fail if readImage() hasn't been called      
  • Fixed JRW-108 Sometimes reading a DNG image uses the wrong ImageReader 
  • Fixed JRW-109 The test suite has been greatly improved, and some tests added.

v1.0.RC4 - 13/Feb/2006

  • Fixed JRW-92: An additional thumbnail described in the PEF makernote is not supported
  • Fixed JRW-99: jrawio provides to ImageIO.getReaderFormatNames() only uppercase names
  • Fixed JRW-100: ImageIO.getImageReaders(Object) does not return any raw reader if Object is not an ImageInputStream
  • Fixed JRW-103: Added RAWProcessor support

v1.0.RC3 - 6/Feb/2006

  • Fixed JRW-4: Add support for Minolta MRW Format
  • Fixed JRW-63: Cannot load a NEF image (DSC_0208.NEF)
  • Fixed JRW-66: Cannot load a NEF image (DSC_3616.NEF)
  • Fixed JRW-67: Cannot load a NEF image (D1XFARLU_COMP.NEF)
  • Fixed JRW-90: getWidth() and getHeight() for some thumbnails return 0.
  • Fixed JRW-94: Raster for PEF format is not correctly read

v1.0.RC2 - 29/Jan/2006

  • Refactored.
  • Fixed JRW-9: Add support for Pentax PEF format
  • Fixed JRW-78: ImageReader getWidth() and getHeight() return wrong values for CR2 files
  • Fixed JRW-79: Some images have strings filled with zeroes, and this leads to invalid XML
  • Fixed JRW-80: In the XML metadata the tag names are missing
  • Fixed JRW-81: Add support for Pentax *ist D
  • Fixed JRW-82: Add support for Pentax *ist DS
  • Fixed JRW-83: Add support for Pentax *ist DL
  • Fixed JRW-91: XML metadata is often screwed up.
  • Now the reference IDE for development is NetBeans 5.0.

v1.0.RC1 - 5/Jan/2006

  • Refactored.
  • Completed support for XML-based metadata.
  • Fixed JRW-8: Add support for Sony SRF format
  • Fixed JRW-44: Add NEF support for Nikon D200
  • Fixed JRW-53: jrawio must behave as a well-written ImageIO plugin
  • Fixed JRW-54: Metadata for NEF is tweaked
  • Fixed JRW-75: A TIFF 'ascii' tag shorter than 4 bytes can prevent an IFD directory from being loaded
  • Fixed JRW-76: Cannot read a NEF image (20051208-0005.NEF)
  • Fixed JRW-77: Cannot read .NDF images (Nikon dust references)

v0.9.8 - 7/Dec/2005

  • Refactored.
  • Now RAWImageInputStream supports mark() and reset() even if uses BitReaders.<

v0.9.7 - 15/Nov/2005

  • Added support for Canon CRW.
  • Added support for Canon CR2.
  • Completed support for Adobe DNG.
  • Fixed JRW-1: Missing 'lib' directory when checking out from CVS
  • Fixed JRW-2: The jrawio plugin don't load automatically as a imageio plugin should
  • Fixed JRW-5: Add NEF support for Nikon D50        
  • Fixed JRW-47: Add support for Canon EOS 300D    
  • Fixed JRW-48: Add support for Canon EOS 350D    
  • Fixed JRW-49: Add support for Canon EOS D30    
  • Fixed JRW-55: EXIF metadata is not read from CRW files
  • Fixed JRW-60: Add NEF support for Nikon D70s
  • Refactored, common code factorized.
  • TIFF_Tag: now short values are available as rationals as well.
  • PixelLoaders have been refactored - added polimorphic methods to save code.
  • IFD redesigned
  • Added support for Nikon lenses.
  • Added preliminary CRW support.
  • Added NDFPixelLoader.
  • Metadata-related classes are now Serializable.
  • PixelLoaders: Directory is now used in place of IFD in parameters.
  • TIFFReImageReaderSupport: Removed useless stuff.
  • BitReader is now an abstract class, not an interface.
  • Normalization is no more applied to raster data. Raster data loaded in memory is now the exact dump of the sensor.
  • PixelLoaders have been renamed RasterReaders.
  • Now the image raster is created by RasterReaders.

v0.8.4 - 22/Dec/2003

  • Added D2H support.

v0.8.3 - 21/Sep/2003

  • NEFTags LENS_TYPE and FLASH_SETTING fields have been validated.
  • Added literal support in NEFTags for FOCUS_MODE, FLASH_SETTING, LENS_TYPE, LIGHT_TYPE, FLASH_USED.
  • Now NEFTags provides string literal values by means of a single toString() method.
  • Two new methods added to NEFMetadata: getEXIFMetadata() and getMakerNoteMetadata().
  • TIFFMetadata methods getSubMetadata() and getNamedMetadata() should be not used by clients any longer.
  • Added thumbnail support.
